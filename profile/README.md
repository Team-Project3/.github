# STS-ent 🎟️

**STS-ent**에 오신 것을 환영합니다! 영화, 콘서트, 미술관 티켓을 예매할 수 있는 원스톱 플랫폼입니다. 쉽고 편리하게 최고의 이벤트를 경험하세요.

![STS-ent](./assets/logo.png)


## 주요 기능

- 📽️ **영화**: 최신 블록버스터 영화 예매.
- 🎤 **콘서트**: 좋아하는 아티스트의 콘서트 예매.
- 🖼️ **미술관**: 주요 전시회 예약.
- 💳 **안전한 결제**: 빠르고 안전한 결제 처리.
